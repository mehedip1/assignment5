3.What is event bubble?

Ans = We can understand event bubblin as a sequence of call the event handlers when one element is nested in another elements and both the elements have registered listeners for the same event. So beginning from the deepest element to its parents covering all its ancestors on the way to top to bottom calling is performed.

4.What is a callback function and why will you use it?

Ans = If you pass another function as a parameter, it is called a callback function. These callback function are often used in libraries and frameworks, such as the JavaScript applications jQuery and Node.js. They are designed to extend functions and be called back for specific events or circumstances.
